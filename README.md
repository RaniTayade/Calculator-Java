# Simple Calculator (Java - Basic)

## 📌 Description
A beginner-friendly calculator program in Java that can perform addition, subtraction, multiplication, and division without OOP.

## 🛠️ Skills Used
- Core Java (basic)
- Scanner Class
- Conditional Statements

## 🚀 Features
- Perform +, -, *, / operations
- Input validation
- Handles division by zero

## ▶️ How to Run
1. Clone the repository
2. Compile: `javac Calculator.java`
3. Run: `java Calculator`
